# stock-analysis
pythonを用いて株価データを取得し、可視化を行うツール


##データは yfinance を利用して取得し、以下のグラフを生成：

折れ線グラフ（終値）

ローソク足チャート

出来高付きローソク足

移動平均線（30日）

ボリンジャーバンド


##使用モジュール：

pandas

matplotlib

mplfinance

yfinance


##出力結果

実行すると、以下の画像ファイルが生成される：

stock_price1.png ：折れ線グラフ

stock_price2.png ：ローソク足チャート

stock_price3.png ：出来高付きローソク足

stock_price4.png ：移動平均線

stock_price5.png ：ボリンジャーバンド
